## サービス説明文

---

### ✈️ こんな人に向いています

- 世界一周・長期旅行中で、旅の記録を発信したい方
- noteやXに旅行記を書きたいけど、文章を書く時間がない方
- 毎日の出来事をメモしているけど、記事にまとめるのが面倒な方
- 旅しながら発信を続けて、将来的に収益化したい方

---

### 📱 どんなサービスですか？

**Telegramというアプリに送るだけで、毎日自動で旅行記事を作ってくれるボット**をあなた専用に作成します。

**使い方はとてもシンプルです。**

```
① 旅先でTelegramに送る
　　→ 「象が道を歩いてた」
　　→ カフェの写真＋「最高のロケーション」
　　→ 位置情報をポン

② 毎日22時に自動で記事が届く
　　→ note用の旅行記（800〜1300文字）
　　→ X（旧Twitter）用の投稿文（500文字以内）

③ コピペしてnoteに貼るだけ
```

**PCは不要です。スマホだけで完結します。**

**【世界一周83日目・スリランカ】コロンボで食べた安飯と、道を歩く象と、それだけで十分だった話**

朝、宿の近くの食堂に入った。メニューは読めないので適当に指さして頼んだ。出てきたのはスパイスが効いたカレー。値段は日本円で100円くらい。うまかった。朝イチで来たからか、地元のおじさんしかいなかった。観光地化されていない分、変に気を遣わなくていい。

📷 [写真1]

食後、当てもなく歩いていたら、象が道を歩いていた。普通に。信号待ちをしている車の横を、ゆっくりと。驚いて立ち止まったが、周りの人は誰も気にしていない。この国ではそういうものらしい。日本でいう、カラスみたいな感覚なのかもしれない。

📷 [写真2]

昼は少し奥まった路地のカフェを見つけた。観光客向けではなく、地元の人が普通に使っている店。コーヒーを頼んで、しばらくぼーっとしていた。旅に出てから、こういう時間が一番好きだと気づいた。何もしていないのに、何かが満たされる感じ。

夕方、念願のスリランカカレーを食べた。辛くない店を探して3日かかった。ようやく見つけた一皿は、ココナッツミルクがベースでまろやかで、スパイスの香りだけがしっかり残る。辛いものが苦手でも、これは食べられる。むしろ、これが一番うまかった。

📷 [写真3]

完璧な一日ではない。移動で迷ったし、暑くて途中でしんどくなった。それでも、象を見て、うまいものを食べて、知らないカフェでぼーっとできた。それで十分だった。
