# EasyLog はじめてガイド

> 旅の記録を送るだけで、毎日自動で旅行記事を作ってくれるサービスです。

---

## EasyLogでできること

1. **Telegramに写真・文章・場所を送る**だけで記録できる
2. 毎晩22時に、その日の記録をもとに**旅行記事を自動で作成**してくれる
3. **note用の記事**と**X（旧Twitter）用の投稿文**が届く

---

## セットアップ前に準備するもの

セットアップ（初期設定）の前に、以下の3つのアカウントを作っておいてください。
すべて**無料**で使えます。

---

### ① Renderのアカウント

サービスを動かすためのサーバーです。

1. [https://render.com](https://render.com) を開く
2. 右上の **Get Started for Free** をクリック
3. **GitHub** または **Google** のアカウントでログイン
4. アカウントが作れたらOK ✅

---

### ② Supabaseのアカウント

記録した写真・文章を保存するデータベースです。

1. [https://supabase.com](https://supabase.com) を開く
2. 右上の **Start your project** をクリック
3. **GitHub** または **Google** のアカウントでログイン
4. アカウントが作れたらOK ✅

---

### ③ GoogleアカウントでGemini APIキーを取得する

記事を作るためのAI（Gemini）を使うために必要です。

#### APIキーの取得

1. [https://aistudio.google.com](https://aistudio.google.com) を開く
2. 右上の **Sign in** から**Googleアカウント**でログイン
3. 左メニューの **Get API key** をクリック
4. **Create API key** をクリック
5. 表示された文字列（APIキー）をどこかにコピーしておく
   ```
   例: AIzaSyDP5AWP5VwZay95ssDugdBOtXb...
   ```

#### 支払い情報の登録（無料で使えます）

> APIを使うには支払い情報の登録が必要です。登録するだけで、通常の使い方では**費用はほぼかかりません**（月数十円〜数百円程度）。

1. [https://console.cloud.google.com](https://console.cloud.google.com) を開く
2. 同じGoogleアカウントでログイン
3. 上部のメニューから **お支払い** を選択
4. **お支払いアカウントを作成** をクリック
5. クレジットカード情報を入力して登録

登録が完了したらOK ✅

---

## 準備ができたら：担当者を招待する

アカウントが作れたら、担当者がセットアップできるよう**2つのサービスに担当者を招待**します。
パスワードを教える必要はありません。招待機能を使います。

---

### RenderにKに担当者を招待する

1. [https://render.com](https://render.com) を開いて自分のアカウントでログイン
2. 右上のアイコン（人のマーク）をクリック
3. **Account Settings** を選択
4. 左メニューの **Team** をクリック
5. **Invite Team Member** をクリック
6. 担当者のメールアドレスを入力して **Send Invite** をクリック

担当者にメールが届くので、あとは担当者が作業します ✅

> セットアップ完了後は、同じ画面から担当者を削除してください。

---

### Supabaseに担当者を招待する

1. [https://supabase.com](https://supabase.com) を開いて自分のアカウントでログイン
2. 左上の組織名（最初は **Personal** と表示されている）をクリック
3. **Organization Settings** を選択
4. 上部メニューの **Members** をクリック
5. **Invite members** をクリック
6. 担当者のメールアドレスを入力して **Send invite** をクリック

担当者にメールが届くので、あとは担当者が作業します ✅

> セットアップ完了後は、同じ画面から担当者を削除してください。

---

### 担当者への連絡

招待が完了したら、担当者に以下を連絡してください。

```
・Renderへの招待を送りました
・Supabaseへの招待を送りました
・GeminiのAPIキー：AIzaSy〇〇〇〇〇〇〇〇（コピーしたもの）
・Geminiの支払い情報の登録が完了した
・旅の開始日：2026-01-28（自分の開始日を記入）
```

これを受け取ったら、セットアップ担当者が設定を進めます。

## EasyLogの使い方

セットアップが完了したら、あとは**Telegramに送るだけ**です。

---

### 文章を記録する

旅先での出来事をそのまま送るだけでOKです。

```
例：
「象が道を歩いてた」
「朝ごはん、ホテルの素朴な料理だったけど美味しかった」
「バスが激混みで死にそうだった」
```

送ると → 「記録しました！」と返ってきます。

---

### 写真を記録する

写真を送るときは、**必ずキャプション（説明文）をつけてください**。
キャプションがないと、AIが写真の内容をわからず記事に反映できません。

**送り方（iPhone・Androidどちらも同じ）**

1. Telegramを開いてBotのチャットへ
2. 左下の 📎 ボタンをタップ
3. **ギャラリー** から写真を選ぶ
4. **送信ボタンの前に**、下のテキスト欄にキャプションを入力する
5. 送信する

```
キャプションの例：
「念願のスリランカカレー、辛くなくて最高だった」
「街中に普通に象がいてびびった」
「最高のロケーションのカフェ見つけた」
```

送ると → 「画像を記録しました！」と返ってきます。

---

### 位置情報を記録する

今いる場所をGoogleマップのリンクとして記事に入れられます。

1. Telegramを開いてBotのチャットへ
2. 左下の 📎 ボタンをタップ
3. **位置情報** を選択
4. 現在地または任意の場所を選んで送信

送ると → 「位置情報を記録しました！」と返ってきます。

---

### 記事を手動で生成する

通常は**毎日22:00に自動で記事が届きます**。
それ以外のタイミングで記事を作りたいときは、Botに以下を送ってください。

```
/generate
```

しばらくすると記事が届きます（1〜2分かかることがあります）。

---

## 届く記事の使い方

記事が届くと、以下の順番でメッセージが送られてきます。

```
① 写真（送った写真が番号付きで届く）
② note用の旅行記事
③ X（旧Twitter）用の投稿文
```

---

### noteに貼り付ける方法

1. [https://note.com](https://note.com) を開いて新規記事を作成
2. Telegramで届いた**②の記事本文**をそのままコピーして貼り付ける
3. 記事の中に `📷 [写真1]` などが書かれている場所に、**①で届いた写真**をアップロードして挿入する
4. 内容を確認して公開する

---

### Xに投稿する方法

1. Telegramで届いた**③のX用テキスト**をコピーする
2. X（旧Twitter）を開いて投稿欄に貼り付ける
3. 写真を追加したい場合は一緒に添付する
4. 投稿する

## よくある質問

**Q. PCがなくてもスマホだけで使えますか？**
A. はい、スマホだけで完結します。

**Q. 毎日投稿しないといけませんか？**
A. いいえ。投稿がない日は「本日の投稿がありません」とだけ届きます。

**Q. 写真を送ったのに記事に反映されていません**
A. キャプション（説明文）をつけて送ってください。キャプションがないとAIが内容を把握できません。

**Q. 記事の内容がおかしいです**
A. 担当者に連絡してください。
